# Simple Reverse Text

文字列を逆さまにする簡単なJavaScriptツールです。

## インストール方法

```bash
npm install simple-reverse-text
```

## 使い方

```javascript
const reverseText = require('simple-reverse-text');

// 文字列を逆さまにする
console.log(reverseText('hello')); // => 'olleh'
console.log(reverseText('こんにちは')); // => 'はちにんこ'
```

## テスト

